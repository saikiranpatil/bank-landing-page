# Bank Landing Page

A modern, responsive landing page for a fictional bank built with **React + Vite + Tailwind CSS**.

## Live Demo
🌐 [View Live Site](https://bank-landing-page-sai-patil.netlify.app/)

## Features
- Clean & professional UI design  
- Fully responsive (mobile-first)  
- Smooth animations & hover effects  
- Hero section, services, testimonials, contact form  
- Built with modern tools for fast development

## Quick Start

1. **Clone the repo**
   ```bash
   git clone https://github.com/saikiranpatil/bank-landing-page.git
   cd bank-landing-page
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

Open **http://localhost:5173** in your browser.

To build for production:
```bash
npm run build
```

## Tech Stack
- **Frontend**: React 18 + Vite
- **Styling**: Tailwind CSS + PostCSS
- **Build Tool**: Vite
